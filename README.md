# Arnold — Portfolio

Personal portfolio built with React, TypeScript and Vite. Dark theme, lime accent, lots of motion.

## Stack

- React 19 + TypeScript + Vite
- Tailwind CSS v4
- GSAP (ScrollTrigger) & Framer Motion
- React Three Fiber + Three.js
- Lenis (smooth scroll)

## Run

```bash
npm install
npm run dev      # http://localhost:5173
npm run build    # production build
```

## Edit content

All personal data (bio, projects, skills, experience, socials) lives in `src/data/portfolio.ts`.
